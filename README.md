## How do I install these formulae?

`brew install felipesere/tap/<formula>`

Or `brew tap felipesere/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "felipesere/tap"
brew "<formula>"
```
